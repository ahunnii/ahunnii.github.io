United Share

A mobile application for good samaritians who want to help the homeless in their area. Users are able to find the homeless based on anynomous tips through a picture and/or description. Users are then able to donate whatever they need and spread the word.

Technology

United Share is built using Apache Cordova on the Ionic platform that uses the Angular.js framework to build cross-platform mobile applications. On the backend United Share uses a Firebase app to handle requests and authenticiation, while a remote noSQL database handles the data.
